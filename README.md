# DigiToken

A digital food token distribution system for college events.

## Overview

DigiToken is a web application designed to digitize food token distribution for college events like fests, freshers, or farewell. It replaces traditional physical tokens with a QR-code-based system.

## Features

- **QR Generation**: Generate unique QR codes for students based on course and year
- **QR Distribution**: Distribute QR codes via email or download for printing
- **QR Verification**: Scan and verify QR codes at food counters
- **Admin Dashboard**: Monitor QR usage statistics
- **Event Management**: Create and manage multiple events

## Tech Stack

- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB
- **QR Code**: qrcode package
- **Email**: Nodemailer
